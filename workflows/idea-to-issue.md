# Idea To Issue Workflow

Use this when an idea is not ready to build yet.

## Goal

Capture ideas clearly so they can become real projects later.

## Issue Format

```md
## Idea

What is the idea?

## Why It Matters

Why is this useful or interesting?

## First Small Version

What is the smallest version worth building?

## Possible Later Versions

What could be added after version one?

## Open Questions

What needs to be decided before building?
```

## Codex Prompt

```text
Turn this idea into a GitHub issue using my idea-to-issue workflow. Keep the first version small and practical.
```
