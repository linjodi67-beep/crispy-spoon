# Codex Workflows

Personal workflow system for using Codex with GitHub.

This repository collects:

- workspace rules for Codex-generated projects
- repeatable project workflows
- GitHub issue and pull request templates
- notes that help turn ideas into tracked work

## Start Here

Use this repository as the control center for future Codex-assisted work.

Recommended flow:

1. Capture a project idea as an issue.
2. Define the first small version.
3. Build the project in `C:\Users\LINJO\CodexProjects`.
4. Save meaningful changes with GitHub.
5. Use pull requests to review and merge finished work.
