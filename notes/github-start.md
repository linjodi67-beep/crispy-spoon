# GitHub Start Note

GitHub is useful here as a long-term memory and version system for Codex-assisted work.

Use it for:

- storing projects
- tracking ideas as issues
- reviewing changes with pull requests
- keeping reusable workflow documents
- preserving rules and templates

Recommended first habit:

When an idea appears, capture it as an issue before building it.

Recommended second habit:

When a useful version works, save it to GitHub with a clear description.
